# get-sum

## objective

Write a function that reutrns the sum of two numbers. The function should be able to handle both integers and floats.

### request

- method: `GET`
- endpoint: `api/get-sum`

query parameters:

- `a`: number
- `b`: number

example: `api/get-sum?a=1&b=2`

### response

```json
{
  "sum": 3
}
```
